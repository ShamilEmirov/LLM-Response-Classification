# LLM Response Classification

## Project Overview

This project classifies Large Language Model (LLM) responses using Natural Language Processing (NLP) and Machine Learning.

The objective is to predict whether Response A, Response B, or a Tie is the preferred answer.

---

## Technologies

- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn

---

## NLP Techniques

- Text Cleaning
- Tokenization
- Stopword Removal
- Lemmatization
- TF-IDF Vectorization

---

## Machine Learning

Model:

- Logistic Regression

Evaluation Metrics:

- Accuracy
- Log Loss

---

## Project Workflow

1. Loaded the datasets.
2. Combined prompts and responses.
3. Cleaned and preprocessed text.
4. Tokenized and lemmatized text.
5. Extracted TF-IDF features.
6. Trained a Logistic Regression model.
7. Evaluated model performance.
8. Generated predictions for the test dataset.

---

## Results

- Validation Accuracy: **37.25%**
- Validation Log Loss: **1.0896**
